Enum View.ViewAccessibilityMode

    Namespace:

    Accessibility mode for controlling View's Accessible implementation. It is only relevant when deriving custom controls from View directly, as classes derived from CustomView (or any of its subclasses) get the Custom mode by default.

    public enum View.ViewAccessibilityMode

    Fields

    Name Description
    Custom

    Custom accessibility implementation. Overriding View.Accessibility...() will be necessary to provide accessibility support for the View.
    Default

    Default accessibility implementation. Overriding View.Accessibility...() virtual methods will have no effect.
